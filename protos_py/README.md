# sellio-cart-protos

Python gRPC stubs for Sellio Cart Service.

Install from GitHub Packages (replace OWNER/REPO and setup auth):

```bash
pip install sellio-cart-protos --index-url https://YOUR_GITHUB_PACKAGES_PYTHON_INDEX/simple/
```

Usage:

```python
from sellio_cart_protos import cart_pb2, cart_pb2_grpc
```
